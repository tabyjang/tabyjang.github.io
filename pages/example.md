# 첫 번째 게시글

안녕하세요! 이것은 GitHub Pages 정적 블로그의 첫 번째 예제 게시글입니다.

## 소개

이 블로그는 다음과 같은 기술로 구축되었습니다:

- **HTML, CSS, Vanilla JavaScript**: 순수 웹 기술만 사용
- **marked.js**: 마크다운 파싱
- **Prism.js**: 코드 하이라이팅
- **Giscus**: GitHub Discussions 기반 댓글 시스템

## 코드 예제

다음은 간단한 JavaScript 예제입니다:

```javascript
function greet(name) {
    return `안녕하세요, ${name}님!`;
}

console.log(greet('방문자'));
```
